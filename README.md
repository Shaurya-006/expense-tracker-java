# Expense Tracker (Java)

## About

This is a simple Java program to track daily expenses. It is a console-based project.

## Features

* Add expense
* View expenses
* Calculate total
* Delete expense

## How to Run

1. Compile:
   javac Main.java Expense.java

2. Run:
   java Main

## Files

* Main.java
* Expense.java
* data.txt

## Note

This project is made for learning basic Java concepts.
