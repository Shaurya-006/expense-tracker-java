# Expense Tracker (Java)

## About the Project

This project is a basic expense tracker developed using Java.
It helps users manage their daily expenses by storing and displaying expense details through a simple console interface.

The main aim of this project is to understand basic Java concepts like classes, objects, and program structure.

---

## Project Structure

Expense-Tracker/

* app/ → contains main program
* model/ → contains expense class
* service/ → handles operations
* util/ → helper functions

---

## Features

* Add new expense
* View all expenses
* Calculate total expenses
* Simple menu-driven program

---

## How to Run

1. Open the project in any Java IDE (VS Code / Eclipse)

2. Compile the program:
   javac app/Main.java

3. Run the program:
   java app.Main

---

## Output

The program shows a menu where the user can select options to manage expenses.
